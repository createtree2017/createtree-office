# createTree Office

사내 업무용 및 매뉴얼 사이트 (createTree Office)

## 프로젝트 개요
- **플랫폼**: 사내 인수인계 매뉴얼, 업무 효율화, 직원 전용 포털
- **기술 스택**:
  - Frontend: React + TypeScript + Vite + TanStack Query
  - UI: Tailwind CSS + shadcn/ui
  - Backend: Express.js + TypeScript
  - Database: PostgreSQL (Neon DB + Drizzle ORM)
  - Editor: Tiptap (Rich Text)
  - Auth: JWT 인증 기반

## 시작하기

### 설치
```bash
npm install
```

### 실행
```bash
npm run dev
```

## 디렉토리 구조
- `client/`: 프론트엔드 (React)
- `server/`: 백엔드 (Express)
- `shared/`: 공용 타입 및 스키마
- `docs/`: 프로젝트 문서 및 설계서
- `db/`: 데이터베이스 스키마 및 마이그레이션
