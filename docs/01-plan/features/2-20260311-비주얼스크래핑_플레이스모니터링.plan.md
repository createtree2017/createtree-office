# 비주얼 스크래핑 모니터링 계획서

> **작성일**: 2026-03-11
> **분류**: Major Feature
> **핵심**: Playwright(스크린샷) + Gemini 2.5 Flash Vision(이미지→데이터)으로 카카오맵/구글 리뷰 수집
> **기존 API 방식 실패 사유**: 카카오맵 서버사이드 API 차단 (2025.10~), Outscraper 로컬 네트워크 타임아웃

---

## 1. 아키텍처 설계

### 기존 파이프라인 (변경 없음)

```
수집(Collector) → 감성분석(SentimentAnalyzer) → 보고서(ReportGenerator) → 알림(Telegram)
```

### 비주얼 스크래핑이 담당하는 부분: **수집(Collector)만 교체**

```
[기존] placeCrawler.crawlKakaoMap() → API 호출 → 404 차단 ❌
[신규] visionCrawler.crawlByVision() → 브라우저 스크린샷 → Gemini Vision → PostData[] ✅
```

> **핵심**: Vision 크롤러는 `PostData[]`를 반환하므로, 기존 감성분석·보고서·알림 파이프라인이 **수정 없이 그대로 동작**합니다.

### 데이터 흐름

```
1. Playwright로 카카오맵/구글맵 리뷰 페이지 접속
2. 첫 화면 1장 스크린샷 (1920×1080)
3. Gemini 2.5 Flash Vision API에 이미지 전송
4. AI가 리뷰 데이터를 JSON으로 추출
5. JSON → PostData[] 변환
6. 기존 파이프라인 진입 (감성분석 → 보고서 → 알림)
```

### 2.2 Gemini Vision 프롬프트 설계

```
당신은 온라인 리뷰 데이터 추출 전문가입니다.
이 이미지는 {platform}의 매장 리뷰 화면 스크린샷입니다.

이미지에 보이는 모든 리뷰를 아래 JSON 형식으로 추출하세요:

{
  "reviews": [
    {
      "author": "작성자 닉네임",
      "rating": 5,             // 별점 (1~5, 없으면 null)
      "content": "리뷰 본문 전체 텍스트",
      "date": "2026.03.10",    // 작성일 (보이는 그대로)
      "hasPhoto": true         // 사진 포함 여부
    }
  ],
  "placeName": "매장 이름",
  "totalReviewCount": "전체 리뷰 수 (보이면)",
  "averageRating": 4.5
}

주의사항:
- 이미지에 실제로 보이는 리뷰만 추출하세요 (추측 금지)
- 별점은 채워진 별(★)의 개수를 정확히 세세요
- 글씨가 잘리거나 안 보이면 "[판독불가]"로 표시
- 리뷰가 없으면 빈 배열로 반환
```

```bash
npm install playwright
npx playwright install chromium  # Chromium 브라우저만 설치 (~150MB)
```

## 3. 카카오맵/구글맵 페이지별 캡처 전략

### 카카오맵

| 항목         | 설명                                    |
| ------------ | --------------------------------------- |
| 접속 URL     | `https://place.map.kakao.com/{placeId}` |
| 대기         | 페이지 로드 후 2~3초 (리뷰 렌더링 대기) |
| 캡처 범위    | viewport 1920×1080, 첫 화면 1장         |
| 예상 리뷰 수 | 첫 화면에 3~5개                         |

### 구글 플레이스

| 항목         | 설명                                    |
| ------------ | --------------------------------------- |
| 접속 URL     | `https://www.google.com/maps/place/...` |
| 대기         | 리뷰 탭 클릭 후 3초                     |
| 캡처 범위    | 리뷰 패널 영역 1장                      |
| 예상 리뷰 수 | 첫 화면에 3~5개                         |
