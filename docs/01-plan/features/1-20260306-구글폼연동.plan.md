# 계획서: 구글 폼 연동 시스템 구축

- **작성일**: 2026-03-06
- **상태**: 계획(Plan)

## 1. 목적
- 관리자가 구글 드라이브에서 만든 설문지/신청폼을 사이트 내 특정 페이지(강좌 신청 등)에 쉽게 연결하여 운영 효율성 증대.

## 2. 주요 기능
- 구글 드라이브 특정 폴더 내 폼 목록 실시간 조회 (GAS API 활용).
- 관리자 페이지 내 폼 선택 및 저장 UI.
- 사용자 페이지 내 구글 폼 테마에 맞춘 iframe 자동 렌더링.

## 3. 기술적 접근
- **Infra**: Google Apps Script (Web App 배포).
- **Storage**: PostgreSQL (`manuals` 테이블 확장).
- **Frontend**: API Fetch & Conditional Rendering.

## 4. 기대 효과
- 개발자 개입 없이 관리자가 직접 신청 양식 변경 가능.
- 데이터 수집을 구글 폼으로 단일화하여 데이터 관리 용이.
